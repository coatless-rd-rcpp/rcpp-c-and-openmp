# RcppCandOpenMP: Using OpenMP with a C Routine and Rcpp

Provides an implementation of incorporating parallelization using
'OpenMP' with 'C' and exporting it into 'R' using 'Rcpp'.

## Details

Disclaimer. Work contained within package is part of an R&D effort to
show a minimal example of how something can be achieved. Care should be
used when transferring it into production.
